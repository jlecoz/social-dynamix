# Tridium

Next.js + React starter site for Tridium Studios with:

- Landing page (`/`)
- Contact page (`/contact`)
- Easy logo/theme customization from one file

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Customize branding and colors

Edit `config/siteConfig.js`:

- `brand.logoText` controls the header wordmark next to the logo mark.
- `theme.*` controls the site color palette.
- `contactEmail` controls contact links and form action.
